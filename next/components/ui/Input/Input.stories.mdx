# Input

Styled Input, that is used as a input throughout CityLibrary web app.

## Props

Input accepts all default HTMLInputProps in addition to custom Props that are as follows:

- <b>hasError</b>: Boolean value, that indicates, that the input should have error styling (red
  border)
- <b>iconLeft</b>: Left Accessory Icon
- <b>iconRight</b>: Right Accessory Icon
- <b>labelContent</b>: string that is positioned on the top of the Input element
- <b>helpText</b>: string that is positioned on the bottom of the Input element
- <b>inputClassName</b>: classname of the input element
