# NewsLetter

NewsLetter

## Props

NewsLetter accepts custom Props that are as follows:

- <b>className</b>: modifies or extends the styles for the component
- <b>title</b>: String that represents the title of the newsletter
- <b>inputPlaceholder</b>: Placeholder for Input element
- <b>buttonContent</b>: String that is written inside the button
- <b>checkboxContent</b>: String that is written inside the Checkbox component
